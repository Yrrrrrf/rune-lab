Complete rune-lab v0.5 target structure: core kernel with forge layer, ui adapter with define layer, four plugins plus future showcase plugin all sharing one standard shape, and the lab app reduced to a thin consumer.

core ← ui ← plugins ← apps requires: kernel-mediated plugin deps mod.ts = export-only, never imported internally
